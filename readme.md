## 🛠️ Installation Steps

### 1. Clone repository

```
git clone https://github.com/wallacemt/top_interviews.git
```

### 2. Go into directory

```
cd daily-leetcode
```

### 3. Execute visual studio code

```
code .
```

### 4. Or execute whatever you want in console

```
gcc -o /solutions/sortMatrixByDiagonals main.c
```
